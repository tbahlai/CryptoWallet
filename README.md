# CryptoWallet

Crypto Wallet Application<br />
Application for viewing information about cryptocurrencies<br />

## Onboarding<br />
When the application is launched for the first time, a three-step onboarding process is displayed, after which the HomeScreen of the program opens.<br />
After a restart application, the onboarding information is not displayed.<br /><br />

## Profile Screen<br />

The profile picture was uploaded from this link: https://thispersondoesnotexist.com/image<br />
The news list is downloaded, and updated every 15 seconds (in chronological order, the first 10 are enough) from this resource: https://rapidapi.com/apidojo/api/ms-finance<br />
Other information on the profile page is static, the buttons are clickable, but do not perform any functionality<br />
The Profile Screen looks like the design. [Profile Screen Design](https://www.figma.com/file/VuemyIDRhyVi2zsdFfy83F/PlantIn-Test-task?node-id=0%3A78&t=a8fp1uiYH1W7BuUG-4)<br />
